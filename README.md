# Cryptography Assignments

### Assignment 1: Playfair Cipher
Cài đặt thuật toán mã hóa và giải mã văn bản bằng phương pháp Playfair.

### Assignment 2: Extended Euclidean Algorithm in GF(2^10)
Cài đặt thuật toán Euclidean mở rộng để tìm nghịch đảo nhân trong trường hữu hạn GF(2^10) với đa thức tối giản m(x) = x^10 + x^3 + 1. 
Yêu cầu hiển thị chi tiết các bước tính toán và các giá trị trung gian.
